# 元素显示模式是

元素（标签）以什么方式进行显示 ，比如<div>独占一行 <span>可以放多个

HTML元素 一般分为块元素 和行内元素 两种类型

# 块元素

常见的块元素有<h1> -<h6> <p> <div> <ol> <li> 其中<div>是典型的块元素

特点

1.比较霸道自己独占一行

2.高度 宽度外边距以及内边距都可以控制

3.宽度默认是一个容器（父级宽度）的100%

是一个容器及盒子里面可以放行内或者块级元素

# 行内元素

常见的行内元素有<a> <strong> <b> <em> <del> <s> <ins> <u> <spam>等其中<span>标签是最典型的行内元素 有的地方也将行内元素称为 内链元素

特点

1.相邻行内元素在一行上 一行可以显示多个

2.高度和宽度直接设置是无效的

3.默认宽度就是它本身内容的宽度

4.行内元素只能容纳文本或其他行内元素

# 行内块元素

<img />  <input/> <td/>

# 元素显示模式转换

把行内元素转化为块级元素 display:block

a{
    把a元素转化为块级元素
    display:block;
}

把块级元素转化为行内元素 display:inline; 把<div>转化为行内元素

转化为行内块元素 display：inline-block； <span>

导航栏小技巧 让文字的行高等于盒子的高度 就可以让文字在当前盒子内垂直居中

line-height: ; 盒子的高度
